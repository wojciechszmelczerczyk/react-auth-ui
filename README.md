## To run

Clone repository.

```sh
git clone https://github.com/wojciechszmelczerczyk/react-auth-ui.git
```

Navigate to project directory.

```sh
cd ./react-auth-ui
```

Install dependencies.

```sh
npm i
```

Run server.

```sh
npm run start
```
